## Problem
This problem comes from [Project Euler 9](https://projecteuler.net/problem=9)


> **Problem**
>
> A Pythagorean triplet is a set of three natural numbers, $$ a < b < c$$, for which,
>
> <div class="text-center py-3">$$ a^2 + b^2 = c^2$$</div>
>
> For example, $$ 3^2 + 4^2 = 9 + 16 = 25 = 5^2$$ 
>
> There exists exactly one Pythagorean triplet for which <div class="text-center py-3">$$ a + b + c = 1000$$</div>
>
> Find the product $$ a \times b \times c$$ 


## Solution

In order to generate Pythagorean Triples we can use [**Euclid's Formula**](https://en.wikipedia.org/wiki/Pythagorean_triple#Generating_a_triple), where $$ a, b, c$$ in the Pythagorean Theorem are

<div class="text-center py-3">$$ a = m^2 - n^2$$</div>
<div class="text-center py-3">$$ b = 2mn$$</div>
<div class="text-center py-3">$$ c = m^2 + n^2$$</div>

And $$ m $$ and $$ n $$ are arbitrary integers where $$ m > n > 0 $$ .

Now, we _could_ go forward as is and write the following code

```python
for n in range (1, 1000):
    for m in range(n+1, 1000):
        a = m**2 - n**2
        b = 2*m*n
        c = m**2 + n**2

        if (a+b+c == 1000):
            print(a*b*c)
            break
```

And it would work fine! But we can make an optimization. We know that we are aiming for

<div class="text-center py-3">$$ a + b + c = 1000$$</div>

So why not perform some simplications? Substituting in Euclid's Formula we get

<div class="text-center py-3">$$ m^2 - n^2 + 2mn + m^2 + n^2 = 1000$$</div>

And simplifying the above equation

<div class="text-center py-3">$$ m(m+n) = 500$$</div>

Finding when $$ m(m+n) = 500$$ and _then_ calculating $$ a $$, $$ b $$, and $$ c $$ is less computationally expensive and faster than the code shown above.

```python
# Project Euler: Problem 9
# Special Pythagorean Triplet

# Euclid's Formula is used here. 
# Check out https://en.wikipedia.org/wiki/Pythagorean_triple

for n in range (1, 500):
    for m in range(n+1, 500):
        if (m*(m+n) == 500):
            a = m**2 - n**2
            b = 2*m*n
            c = m**2 + n**2

            print(a*b*c)
            break
```

// Project Euler: Problem 9

// Alternate Rust version

fn main() {
    'outer: for n in 1_i32..=500 {
        for m in (n+1)..=500 {
            if m*(m+n) == 500 {
                let a: i32 = m.pow(2) - n.pow(2);
                let b: i32 = 2*m*n;
                let c: i32 = m.pow(2) + n.pow(2);

                println!("{}", a*b*c);
                break 'outer;
            }
        }
    }
}
